time_library module¶

Librería propia que recoge las funciones de tiempo usadas a la hora de hacer cálculos sobre timestamps o inserciones en la base de datos.

time_library.get_timestamp()¶

Función que devuelve el timestamp actual

Devuelve

timestamp en formato («%H:%M:%S %Y-%m-%d»)

time_library.hour_difference(t1, t2)¶

Función que devuelve la diferencia en horas entre dos timestamps

Parámetros

  • t1 – primer timestamp

  • t2 – segundo timestamp

Devuelve

int con las horas de diferencia entre t1 y t2

time_library.valid_time_difference(t1, t2)¶

Función para comprobar si la diferencia de tiempo es válida (> 1 minuto)

Parámetros

  • t1 – primer timestamp

  • t2 – segundo timestamp

Devuelve

true si ha pasado más de 1 minuto, false si no
